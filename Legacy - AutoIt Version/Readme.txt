Always on Top - Readme

Created with AutoIt v3.

This application sits in the notification tray.

To use Always on Top, click the Always on Top icon in the notification tray and click "Always on Top". Then click the window you want on top.

When it is successful, the title of the window will be appended with " - Always on Top"

Full source is available in AoT.au3

Compatible with Windows 7 - Windows 10 and all versions in between, both 32 and 64 bit. Untested on Windows Server.
